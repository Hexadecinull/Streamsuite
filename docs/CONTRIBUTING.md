\# Contributing to StreamSuite



We welcome contributions! Please follow these guidelines.



\## Code Style

\- \*\*No inline comments\*\* — code should be self-documenting via clear naming.

\- All PHP and JS files must include the GPL-3.0 license header.

\- Use 4 spaces for indentation.

\- Functions do one thing and have descriptive names.



\## Pull Request Process

1\. Fork the repository.

2\. Create a feature branch.

3\. Make changes, ensure linting passes.

4\. Submit a PR with a clear description.



\## Reporting Issues

Use GitHub Issues. Include steps to reproduce and environment details.



\## License

By contributing, you agree that your code will be licensed under GPL-3.0.
