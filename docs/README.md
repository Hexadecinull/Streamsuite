\# StreamSuite



\*\*Stream everything. Own nothing. Pay nothing.\*\*



StreamSuite is a free, open-source streaming website that requires no accounts, shows no ads, and can be self-hosted on any LAMP stack. It uses TMDB for metadata and public embed providers for video.



\## Features

\- No account required — guest token persists history and favorites

\- Zero ads, zero tracking

\- 5 built-in themes (Obsidian, Midnight, Forest, Ember, Paper)

\- Continue watching across sessions

\- Full TV series support with season/episode navigation

\- Server switching for video embeds

\- Self-hosted fonts and icons (no Google Fonts)

\- GPL-3.0 licensed



\## Tech Stack

\- PHP 8.1+ (no framework)

\- Vanilla JavaScript (ES2022)

\- CSS3 with CSS variables

\- MySQL/MariaDB
